# TER (Tres en raya)

Juego clasico de Tres en raya con fichas movibles (Papa vs Neo).

Demo en vivo: https://vict0rtilla.github.io/TER/

## Como jugar

- Cada jugador tiene 3 fichas: Papa (X) y Neo (O).
- Fase 1 (colocar): turnos para poner una ficha en un nodo vacio.
- Cuando ambos tienen 3 fichas, empieza la fase 2.
- Fase 2 (mover): selecciona una ficha propia y muevela a un nodo adyacente vacio.
- Gana quien forme una linea de 3.
- Usa el boton "Reiniciar" para empezar otra ronda.
